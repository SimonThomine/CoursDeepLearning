Backbone et fonction de perte#

L‚Äôarticle utilise ResNet18 comme architecture de r√©seau, avec le mod√®le teacher pr√©-entra√Æn√© sur ImageNet. Le student utilise la m√™me architecture, mais son r√©seau est initialis√© al√©atoirement.

Comme illustr√© dans la figure ci-dessus, la fonction de perte est calcul√©e sur les sorties des 3 premiers groupes de couches de ResNet18. class resnet18timm(nn.Module):
    def __init__(self,backbone_name="resnet18",out_indices=[1,2,3],pretrained=True):
        super(resnet18timm, self).__init__()     
        # Features only permet permet de ne r√©cup√©rer que les features et pas la sortie du r√©seau, out_indices permet de choisir les couches √† r√©cup√©rer
        self.feature_extractor = timm.create_model(backbone_name,pretrained=pretrained,features_only=True,out_indices=out_indices)
        if pretrained:

            self.feature_extractor.eval() 
            for param in self.feature_extractor.parameters():
                param.requires_grad = False   
        
    def forward(self, x):
        features = self.feature_extractor(x)
        return features

Fonction de perte#

La fonction de perte utilise la distance euclidienne (MSE), d√©finie comme suit : \(D(I_1, I_2) = \sqrt{\sum_{i=1}^{m} \sum_{j=1}^{n} \left( I_1(i,j) - I_2(i,j) \right)^2}\) o√π \(I_1\) et \(I_2\) sont nos deux images.

Notre impl√©mentation de la perte utilise cette distance pour comparer les feature maps entre elles pour les 3 paires de feature maps :

class loss_kdad:
  def __init__(self):
    pass
  # fs_list : liste des features du student et ft_list : liste des features du teacher
  def __call__(self,fs_list, ft_list):
    t_loss = 0
    N = len(fs_list)
    for i in range(N):
      fs = fs_list[i]
      ft = ft_list[i]
      _, _, h, w = fs.shape
      # Normaliser les features am√©liore les r√©sultats
      fs_norm = F.normalize(fs, p=2)
      ft_norm = F.normalize(ft, p=2)

      # Calcul de la distance euclidienne
      f_loss = 0.5 * (ft_norm - fs_norm) ** 2
      # On prend la moyenne de la loss sur tous les pixels
      f_loss = f_loss.sum() / (h * w)
      t_loss += f_loss

    return t_loss / N
