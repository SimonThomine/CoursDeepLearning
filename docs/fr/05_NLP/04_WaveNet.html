Cr√©ation du mod√®le et entra√Ænement#

Pour commencer, nous allons r√©implementer le mod√®le du notebook pr√©c√©dent, mais avec PyTorch.

import torch 
import torch.nn as nn
import torch.nn.functional as F

class fcn(nn.Module):
  def __init__(self,embed_dim=10,context_len=8,hidden_dim=300, *args, **kwargs) -> None:
    super().__init__(*args, **kwargs)
    self.embed_dim=embed_dim
    self.context_len=context_len
    
    #La fonction nn.Embedding de pytorch est l'√©quivalent de la matrice C 
    self.embedding=nn.Embedding(46,embed_dim)
    self.layer1=nn.Linear(embed_dim*context_len,hidden_dim)
    self.layer2=nn.Linear(hidden_dim,46)

  def forward(self,x):       
    embed=self.embedding(x) # Remplace la matrice C
    embed=embed.view(-1,self.embed_dim*self.context_len)
    hidden=F.tanh(self.layer1(embed))
    logits=self.layer2(hidden)
    return logits

model=fcn(context_len=8)
epochs=50
lr=0.2
optimizer=torch.optim.SGD(model.parameters(),lr=lr)
for p in model.parameters():
  p.requires_grad = True

Comment g√©rer l‚Äôarchitecture hi√©rarchique#

Ce qu‚Äôon veut dans notre mod√®le, c‚Äôest traiter en parall√®le des groupes d‚Äôembedding en regroupant les caract√®res cons√©cutifs. Sur PyTorch, si on fait passer un tenseur de taille \(B \times L \times C\) dans une couche lin√©aire de taille \(C \times H\), on obtient un tenseur de taille \(B \times L \times H\), et c‚Äôest exactement ce que l‚Äôon veut pour impl√©menter le r√©seau WaveNet.

Maintenant, il faut trouver comment modifier la taille du tenseur pour faire les op√©rations du WaveNet. Nos 8 embedding sont regroup√©s par deux puis trait√©s en parall√®le. √Ä la couche suivante, ils sont √† nouveau regroup√©s par deux. Donc √† chaque √©tape, on double la taille \(H\) (ou \(C\)) et on divise par deux \(L\).

Pour √™tre plus clair, √† la premi√®re √©tape, nous avons un tenseur de taille \(B \times 8 \times 10\) que l‚Äôon veut transformer en tenseur de taille \(B \times 4 \times 20\). On peut impl√©menter √ßa avec view() de PyTorch.

Essayons de formaliser √ßa avec une couche que l‚Äôon pourra utiliser dans notre r√©seau :

class FlattenConsecutive(nn.Module):
  # n est le facteur de regroupement (toujours 2 pour nous)
  def __init__(self, n):
    super(FlattenConsecutive, self).__init__()
    self.n = n   
  def __call__(self, x):
    # On r√©cup√®re les dimensions de l'entr√©e
    B, T, C = x.shape 
    # On fait la transformation x2 et /2
    x = x.view(B, T//self.n, C*self.n)
    if x.shape[1] == 1: 
      x = x.squeeze(1) # Si le tensor a une dimension qui vaut 1, on la supprime
    self.out = x
    return self.out

Cr√©ation du mod√®le#

Il est temps de cr√©er notre mod√®le. Pour plus de simplicit√©, on utilise nn.Sequential pour regrouper nos couches.

class wavenet(nn.Module):
  def __init__(self,embed_dim=10,hidden_dim=128, *args, **kwargs) -> None:
    super().__init__(*args, **kwargs)
    
    self.net=nn.Sequential(nn.Embedding(46,embed_dim),
      # B*8*10
      FlattenConsecutive(2), nn.Linear(embed_dim*2,hidden_dim),nn.Tanh(),
      # B*4*hidden_dim
      FlattenConsecutive(2), nn.Linear(hidden_dim*2,hidden_dim),nn.Tanh(),
      # B*2*hidden_dim
      FlattenConsecutive(2), nn.Linear(hidden_dim*2,hidden_dim),nn.Tanh(),
      # B*hidden_dim
      nn.Linear(hidden_dim,46)
    )   
        
  def forward(self,x):
    logits=self.net(x)
    return logits
